# Generate build:
$ `ng build --prod --base-href="./"`

# Angular CLI:
$ `ng --version`
$ `Angular CLI: 11.2.3`

# NodeJS:
$ `node -v`
$ `v15.11.0`

# JSON Server:
$ `npm install -g json-server`
$ `json-server .\db.json`

# Storybook:
$ `npx -p @storybook/cli sb init`
$ `npm run storybook` (port 6006)

# OBS:
$ Caso de erro ao inciar o projeto, deletar o `package-lock.json`.
